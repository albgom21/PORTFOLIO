WILDLESS

Creación de un motor de videojuegos 3D y un pequeño juego desarrollado en él.

Este proyecto se basa en un motor de videojuegos 3D y un juego cargado como DLL en el motor. Se trata de un motor sin interfaz creado a partir de ir integrando distintas librerias en un proyecto de Visual Studio para cada módulo que lo compone: Física, Sonido, Renderizado, UI, Lua. Se utilizan factorías para la creación de los componentes que tiene cada objeto (ECS) que se encuentra en escena, son creados mediante archivos lua. La dependencias del motor se compilan mediante un .bat para facilitar así el proceso.

TECNOLOGÍAS USADAS

C++, OGRE 3D, SDL, Bullet, FMOD, Lua

CONTRIBUCIONES

El proyecto fue desarrollado por un equipo de 9 personas. Mis contribuciones se centran en la integración de librerías en el proyecto, implementación del módulo de render y componentes, manejo de las escenas (maquina de estados) y carga de datos usando Lua. Creación de mecánicas como coger objeto, disparar, destruir y sistema de logros.
